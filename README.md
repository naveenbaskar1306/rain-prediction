
This project predicts whether it will rain or not based on weather conditions using Machine Learning.

## ⚙️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

## 📊 Dataset

The dataset contains:

* Precipitation
* Temperature (Max & Min)
* Wind Speed
* Weather Condition

## 🧠 Model Used

* Logistic Regression (Binary Classification)

## 🔄 Workflow

1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Model Training
5. Model Evaluation
6. Prediction

## 📈 Results

* The model predicts rain or no rain with good accuracy
* Evaluated using accuracy score and confusion matrix

```bash
pip install pandas scikit-learn
python your_file_name.py
```
